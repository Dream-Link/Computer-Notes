🖥️ कंप्यूटर सिस्टम का अवलोकन - हार्डवेयर डिवाइस

🔧 कंप्यूटर सिस्टम की परिभाषा

कंप्यूटर सिस्टम हार्डवेयर और सॉफ्टवेयर का एक संयोजन है जो डेटा को प्रोसेस करने के लिए मिलकर काम करता है।

कंप्यूटर सिस्टम के मुख्य घटक:
🔹 मुख्य घटक:
  • हार्डवेयर (Hardware): भौतिक घटक
  • सॉफ्टवेयर (Software): प्रोग्राम और निर्देश
  • डेटा (Data): जानकारी जो प्रोसेस की जाती है
  • यूजर (User): कंप्यूटर का उपयोग करने वाला व्यक्ति

🖥️ हार्डवेयर डिवाइस - विस्तृत अध्ययन

1. सेंट्रल प्रोसेसिंग यूनिट (CPU)
🔹 CPU की परिभाषा:
  • कंप्यूटर का "दिमाग" कहलाता है
  • सभी गणनाएं और निर्देशों का पालन करता है
  • ALU (Arithmetic Logic Unit) और CU (Control Unit) से मिलकर बना होता है
🔹 CPU के भाग:
  • ALU (Arithmetic Logic Unit): गणितीय और तार्किक संक्रियाएं
  • CU (Control Unit): निर्देशों को नियंत्रित करता है
  • Registers: अस्थायी डेटा स्टोरेज
  • Cache Memory: तेज़ डेटा एक्सेस के लिए
महत्वपूर्ण: CPU की गति GHz (Gigahertz) में मापी जाती है। जितनी अधिक GHz, उतनी तेज़ प्रोसेसिंग।
2. मेमोरी (Memory)
🔹 प्राइमरी मेमोरी:
  • RAM (Random Access Memory): अस्थायी मेमोरी, बिजली बंद होने पर डेटा खो जाता है
  • ROM (Read Only Memory): स्थायी मेमोरी, बिजली बंद होने पर भी डेटा रहता है
  • Cache Memory: CPU और RAM के बीच तेज़ मेमोरी
🔹 RAM के प्रकार:
  • DRAM (Dynamic RAM): सामान्य RAM, रिफ्रेश की आवश्यकता
  • SRAM (Static RAM): तेज़ RAM, रिफ्रेश की आवश्यकता नहीं
  • DDR RAM: Double Data Rate RAM
🔹 ROM के प्रकार:
  • PROM: Programmable ROM
  • EPROM: Erasable Programmable ROM
  • EEPROM: Electrically Erasable Programmable ROM
3. इनपुट डिवाइस (Input Devices)
🔹 मुख्य इनपुट डिवाइस:
  • कीबोर्ड (Keyboard): टेक्स्ट और कमांड इनपुट के लिए
  • माउस (Mouse): पॉइंटिंग और क्लिकिंग के लिए
  • टचपैड (Touchpad): लैपटॉप में माउस का विकल्प
  • माइक्रोफोन (Microphone): ऑडियो इनपुट के लिए
  • वेबकैम (Webcam): वीडियो इनपुट के लिए
  • स्कैनर (Scanner): डॉक्यूमेंट को डिजिटल बनाने के लिए
🔹 विशेष इनपुट डिवाइस:
  • जॉयस्टिक: गेमिंग के लिए
  • लाइट पेन: स्क्रीन पर सीधे लिखने के लिए
  • ट्रैकबॉल: माउस का विकल्प
  • टच स्क्रीन: स्पर्श से इनपुट
4. आउटपुट डिवाइस (Output Devices)
🔹 मुख्य आउटपुट डिवाइस:
  • मॉनिटर (Monitor): विजुअल आउटपुट दिखाने के लिए
  • प्रिंटर (Printer): हार्ड कॉपी प्रिंट करने के लिए
  • स्पीकर (Speaker): ऑडियो आउटपुट के लिए
  • हेडफोन (Headphones): व्यक्तिगत ऑडियो के लिए
🔹 मॉनिटर के प्रकार:
  • CRT (Cathode Ray Tube): पुराना तकनीक
  • LCD (Liquid Crystal Display): कम बिजली खपत
  • LED (Light Emitting Diode): बेहतर चमक
  • OLED: सबसे अच्छी गुणवत्ता
🔹 प्रिंटर के प्रकार:
  • इंकजेट प्रिंटर: स्याही का उपयोग
  • लेजर प्रिंटर: टोनर का उपयोग, तेज़ प्रिंटिंग
  • डॉट मैट्रिक्स प्रिंटर: कार्बन कॉपी के लिए
  • 3D प्रिंटर: त्रिआयामी वस्तुएं बनाने के लिए
5. स्टोरेज डिवाइस (Storage Devices)
🔹 प्राइमरी स्टोरेज:
  • हार्ड डिस्क ड्राइव (HDD): चुंबकीय स्टोरेज
  • सॉलिड स्टेट ड्राइव (SSD): फ्लैश मेमोरी, तेज़ एक्सेस
🔹 सेकेंडरी स्टोरेज:
  • USB फ्लैश ड्राइव: पोर्टेबल स्टोरेज
  • CD/DVD/Blu-ray: ऑप्टिकल स्टोरेज
  • मेमोरी कार्ड: छोटे डिवाइस के लिए
  • एक्सटर्नल हार्ड ड्राइव: बैकअप के लिए
स्टोरेज यूनिट: Bit → Byte → KB → MB → GB → TB → PB
1 Byte = 8 Bits, 1 KB = 1024 Bytes, 1 MB = 1024 KB

🔌 मदरबोर्ड और कनेक्टिविटी

मदरबोर्ड के घटक:
🔹 मुख्य घटक:
  • CPU Socket: प्रोसेसर लगाने के लिए
  • RAM Slots: मेमोरी लगाने के लिए
  • Expansion Slots: अतिरिक्त कार्ड के लिए
  • BIOS/UEFI: बूट प्रोसेस के लिए
  • Chipset: डेटा ट्रांसफर नियंत्रण

💿 सॉफ्टवेयर - ऑपरेटिंग सिस्टम और एप्लीकेशन सॉफ्टवेयर

📋 सॉफ्टवेयर की परिभाषा और वर्गीकरण

सॉफ्टवेयर निर्देशों का एक सेट है जो कंप्यूटर को बताता है कि क्या करना है।

सॉफ्टवेयर के मुख्य प्रकार:
🔹 मुख्य वर्गीकरण:
  • सिस्टम सॉफ्टवेयर: कंप्यूटर हार्डवेयर को नियंत्रित करता है
  • एप्लीकेशन सॉफ्टवेयर: उपयोगकर्ता के काम के लिए
  • प्रोग्रामिंग सॉफ्टवेयर: नए प्रोग्राम बनाने के लिए

🖥️ ऑपरेटिंग सिस्टम (Operating System)

ऑपरेटिंग सिस्टम की परिभाषा:

ऑपरेटिंग सिस्टम एक सिस्टम सॉफ्टवेयर है जो कंप्यूटर हार्डवेयर और उपयोगकर्ता के बीच इंटरफेस का काम करता है।

🔹 OS के मुख्य कार्य:
  • प्रोसेस मैनेजमेंट: प्रोग्राम्स को चलाना और नियंत्रित करना
  • मेमोरी मैनेजमेंट: RAM का उपयोग नियंत्रित करना
  • फाइल सिस्टम मैनेजमेंट: फाइलों को व्यवस्थित करना
  • डिवाइस मैनेजमेंट: हार्डवेयर डिवाइस को नियंत्रित करना
  • सिक्यूरिटी: सिस्टम की सुरक्षा करना
  • यूजर इंटरफेस: उपयोगकर्ता के साथ संपर्क
ऑपरेटिंग सिस्टम के प्रकार:
🔹 उपयोगकर्ता के आधार पर:
  • Single User OS: एक समय में एक उपयोगकर्ता (MS-DOS, Windows 95/98)
  • Multi User OS: एक साथ कई उपयोगकर्ता (Unix, Linux, Windows Server)
🔹 प्रोसेसिंग के आधार पर:
  • Single Tasking: एक समय में एक काम (MS-DOS)
  • Multi Tasking: एक साथ कई काम (Windows, Linux, macOS)
🔹 प्रोसेसर के आधार पर:
  • Single Processing: एक CPU
  • Multi Processing: कई CPU
प्रमुख ऑपरेटिंग सिस्टम:
🔹 डेस्कटॉप OS:
  • Microsoft Windows: सबसे लोकप्रिय (Windows 10, 11)
  • macOS: Apple कंप्यूटर के लिए
  • Linux: ओपन सोर्स (Ubuntu, CentOS, Red Hat)
  • Unix: सर्वर के लिए
🔹 मोबाइल OS:
  • Android: Google द्वारा विकसित
  • iOS: Apple iPhone के लिए
  • Windows Phone: Microsoft (बंद हो गया)
Windows Operating System विस्तार से:
🔹 Windows के संस्करण:
  • Windows 1.0 (1985): पहला GUI आधारित
  • Windows 95/98/ME: उपभोक्ता संस्करण
  • Windows NT/2000/XP: व्यावसायिक संस्करण
  • Windows Vista/7/8/8.1: आधुनिक फीचर्स
  • Windows 10/11: वर्तमान संस्करण
🔹 Windows की विशेषताएं:
  • GUI (Graphical User Interface): आसान उपयोग
  • Multitasking: एक साथ कई प्रोग्राम
  • Plug and Play: हार्डवेयर की आसान पहचान
  • File Management: फाइल एक्सप्लोरर
  • Security Features: Windows Defender

📱 एप्लीकेशन सॉफ्टवेयर

एप्लीकेशन सॉफ्टवेयर की परिभाषा:

एप्लीकेशन सॉफ्टवेयर वे प्रोग्राम हैं जो उपयोगकर्ता के विशिष्ट कार्यों को पूरा करने के लिए बनाए गए हैं।

🔹 एप्लीकेशन सॉफ्टवेयर के प्रकार:
  • वर्ड प्रोसेसिंग: MS Word, Google Docs
  • स्प्रेडशीट: MS Excel, Google Sheets
  • प्रेजेंटेशन: MS PowerPoint, Google Slides
  • डेटाबेस: MS Access, MySQL
  • ग्राफिक्स: Photoshop, CorelDraw
  • मल्टीमीडिया: VLC, Windows Media Player
  • वेब ब्राउज़र: Chrome, Firefox, Edge
  • एंटीवायरस: Norton, McAfee, Avast
सॉफ्टवेयर लाइसेंस के प्रकार:
🔹 मुख्य लाइसेंस प्रकार:
  • Proprietary Software: कॉपीराइट संरक्षित (MS Office)
  • Open Source: मुफ्त और संशोधन योग्य (Linux, LibreOffice)
  • Freeware: मुफ्त उपयोग (Google Chrome)
  • Shareware: परीक्षण अवधि के साथ
  • Trial Software: सीमित समय के लिए मुफ्त

🔧 सिस्टम सॉफ्टवेयर

सिस्टम सॉफ्टवेयर के घटक:
🔹 मुख्य घटक:
  • Device Drivers: हार्डवेयर को नियंत्रित करने के लिए
  • Utilities: सिस्टम रखरखाव के लिए

  • System Libraries: सामान्य फंक्शन्स
यूटिलिटी सॉफ्टवेयर:

  • Disk Defragmenter: हार्ड डिस्क को व्यवस्थित करना

परीक्षा टिप: ऑपरेटिंग सिस्टम के कार्य, विभिन्न OS के नाम, सॉफ्टवेयर के प्रकार और लाइसेंस के बारे में प्रश्न अक्सर पूछे जाते हैं।

🔹 मुख्य विशेषताएं:
  • WYSIWYG: What You See Is What You Get
  • Spell Check: वर्तनी जांच
  • Grammar Check: व्याकरण जांच
  • Auto Correct: स्वचालित सुधार

🔹 Home Tab:
  • Clipboard: Cut, Copy, Paste, Format Painter
  • Font: Font type, size, color, bold, italic
  • Paragraph: Alignment, bullets, numbering
  • Styles: Predefined text styles
  • Editing: Find, Replace, Select
🔹 Insert Tab:
  • Pages: Cover page, blank page, page break
  • Tables: Insert and format tables
  • Illustrations: Pictures, shapes, charts
  • Links: Hyperlinks, bookmarks
  • Header & Footer: Page headers and footers
  • Text: Text box, WordArt, symbols
🔹 Design Tab:
  • Document Formatting: Themes, colors

🔹 Layout Tab:
  • Page Setup: Margins, orientation, size
  • Paragraph: Indentation, spacing
  • Arrange: Text wrapping, positioning

✏️ Text Formatting

Character Formatting:
🔹 Font Formatting:
  • Font Type: Arial, Times New Roman, Calibri

Paragraph Formatting:
🔹 Alignment:
  • Left Align (Ctrl+L): बाएं तरफ संरेखण

🔹 Indentation:
  • Left Indent: बाएं मार्जिन से दूरी
  • Right Indent: दाएं मार्जिन से दूरी

🔹 Line Spacing:
  • Single: एक लाइन की दूरी
  • 1.5 Lines: डेढ़ लाइन की दूरी
  • Double: दो लाइन की दूरी
  • Custom: अपनी पसंद की दूरी

🔹 महत्वपूर्ण घटनाएं:
  • 1969: ARPANET की शुरुआत (अमेरिकी रक्षा विभाग)

  • 1983: ARPANET में TCP/IP का उपयोग
  • 1989: Tim Berners-Lee द्वारा World Wide Web का आविष्कार
  • 1991: पहली वेबसाइट का launch
  • 1993: Mosaic browser का विकास
  • 1995: Commercial internet की शुरुआत
भारत में इंटरनेट:
🔹 भारतीय इंटरनेट इतिहास:
  • 1986: ERNET (Education and Research Network)
  • 1995: VSNL द्वारा commercial internet
  • 1998: Private ISP की अनुमति
  • 2000: Broadband की शुरुआत
  • 2016: Digital India initiative
